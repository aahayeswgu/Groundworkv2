    Main sync effect (`useEffect` depending on `[map, pins, activeStatusFilter, deletePin, onEditPin]`):
    - If `!map` return early

    - Compute `visibleIds` = new Set of visible pin ids
    - Remove stale markers: iterate markerPool.current, for any id NOT in visibleIds → `marker.map = null`, delete from pool
    - For each visible pin:
      - If marker exists in pool AND pin status hasn't changed (compare marker's dataset.status to pin.status):
        - Update `marker.position` to `{ lat: pin.lat, lng: pin.lng }` (handles pin coord updates)
        - Continue
      - If marker exists but status changed: remove old marker (`marker.map = null`, delete from pool)
      - Create new AdvancedMarkerElement:
        ```
        const el = createPinMarkerElement(pin.status)
        el.dataset.status = pin.status   // for change detection
        el.dataset.pinId = pin.id
        const marker = new google.maps.marker.AdvancedMarkerElement({
          position: { lat: pin.lat, lng: pin.lng },
          map,
          content: el,
          title: pin.title,
        })
        ```
      - Attach click listener: `marker.addListener('click', () => handleMarkerClick(pin, marker))`
      - Store in pool: `markerPool.current.set(pin.id, marker)`
    - If openPinId.current is set and its pin was deleted (not in pins array): close info window, clear openPinId

    handleMarkerClick (useCallback, deps: [map, deletePin, onEditPin]):
    ```
    function handleMarkerClick(pin: Pin, marker: google.maps.marker.AdvancedMarkerElement) {
      if (!infoWindow.current) {
        infoWindow.current = new google.maps.InfoWindow()
        infoWindow.current.addListener('closeclick', () => { openPinId.current = null })
      }
      // Toggle behavior (D-10)
      if (openPinId.current === pin.id) {
        infoWindow.current.close()
        openPinId.current = null
        return
      }
      infoWindow.current.setContent(buildInfoWindowContent(pin))
      infoWindow.current.open({ anchor: marker, map })
      openPinId.current = pin.id
    }
    ```
