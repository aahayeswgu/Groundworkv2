has-data detection helper (from research Pattern 7):
```typescript
function dayHasData(plannerDays: Record<string, DayPlan>, dateStr: string): boolean {
  const day = plannerDays[dateStr];
  if (!day) return false;
  return (
    day.stops.length > 0 ||
    day.activityLog.length > 0 ||
    day.notes.some((n) => n.trim().length > 0)
  );
}
```

CRITICAL date construction:
- Always use `new Date(isoStr + "T00:00:00")` to prevent UTC offset shifting date display
- When computing ISO date from year/month/day: `new Date(year, month, day)` local constructor is correct
- `new Date(year, month, day).toLocaleDateString("en-CA")` gives "YYYY-MM-DD" (ISO format in local time)
